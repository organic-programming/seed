# ObservabilityPanel

Public API: `ObservabilityPanel(kit: ObservabilityKit)` renders Settings, Logs, Metrics, and Events tabs.

Customisation: inherits `ThemeData`, `ColorScheme`, text theme, icon theme, and parent layout constraints.

Sizing: flexes to its parent; prefer at least 720x480.

Embedding: place in a full-screen route, drawer destination, or modal sheet.
