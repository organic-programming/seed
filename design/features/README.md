# Features

Features are **cross-cutting design topics** that don't belong to a
single holon. A feature may span multiple SDKs, holons, or layers
of the ecosystem.

Unlike holon design directories (which describe one specific holon),
feature directories describe theoretical, architectural, or central
concerns that affect the platform as a whole.
