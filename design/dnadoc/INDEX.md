# DNADoc — Platform Specifications

Design and evolution of the root-level specification documents
that define the Organic Programming platform:

| Document | Scope |
|---|---|
| `OP.md` | Platform specification — holons, verbs, transports |
| `CONVENTIONS.md` | Standards for holon structure and naming |
| `PROTOCOL.md` | Wire protocol and communication semantics |
| `PROTO.md` | Protobuf conventions and sharing rules |
| `HOLON_YAML.md` | Manifest schema reference |
| `AGENT.md` | LLM agent integration rules |
| `WHY.md` | Philosophy and motivation |
| `README.md` | Entry point and overview |
